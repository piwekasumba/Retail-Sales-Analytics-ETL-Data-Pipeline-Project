• Retail Sales SQL Pipeline
---
 End-to-End ETL Simulation for Data Cleaning, Transformation & Business Reporting
---
• Project Overview
---
- This project demonstrates an end-to-end SQL-based data pipeline that simulates how retail sales data is processed in real-world analytics environments.

- It focuses on transforming raw transactional data into a structured, analysis-ready dataset used for reporting and business decision-making.

- The project reflects core skills required in Data Analyst and Junior Data Engineering roles.

---
• ETL Workflow (Real-World Simulation)
---
• Extract

- Raw retail sales dataset loaded into a relational database from CSV format
---
• Transform

- Handled missing values and duplicate records
- Standardised product names and category fields
- Fixed inconsistent date formatting
- Cleaned transactional inconsistencies
- Structured dataset for analytical use
---
• Load

- Created structured SQL tables optimized for querying and reporting
- Prepared dataset for business analysis workflows
---
• Core SQL Skills Demonstrated

- Data cleaning using SQL
- Joins (INNER, LEFT JOIN)
- Aggregations (SUM, COUNT, AVG, GROUP BY)
- ETL pipeline design thinking
- Data structuring for analytics
- Business reporting logic

- Business reporting and decision support
